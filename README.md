# deduplication  
Linux下对指定文件夹进行文件查重的bash脚本，还可以对找到的重复文件进行硬链接以节约存储空间，配合dingtalkrobot脚本还可以将运行结果发送到钉钉。

## 使用方法
1. 打开脚本，按需求修改头部的几个参数。
2. 运行 deduplication "要查重的文件夹"
3. 耐心等候片刻
#### 如果要查重的文件夹比较多，也可以放在deduplication.list文件中，每行一个，用"#"注释
